# GlintsContainer

`GlintsContainer` is a combination of `BaseContainer` and `PageContainer`.

```
<GlintsContainer>
  <Loading />
</GlintsContainer>
```

## BaseContainer

`BaseContainer` defines common styles that are required by some Aries components. It also sets the font-family to Poppins.

```
<BaseContainer>
  <Loading />
</BaseContainer>
```

## PageContainer

A container that centers its contents on the page. Its width and margins are responsive.

```
<PageContainer>
  <Loading />
</PageContainer>
```
